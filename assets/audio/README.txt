This directory contains audio pronunciation files for the Dzongkha dictionary.

File naming convention:
- Use the Dzongkha word as the filename (e.g., "go.mp3" for the word "go")
- Use lowercase filenames
- For words with spaces, replace spaces with underscores (e.g., "go_tsukken.mp3")

Audio file requirements:
- Format: MP3
- Recommended bitrate: 128kbps
- Keep files small (under 100KB if possible) for better app performance
